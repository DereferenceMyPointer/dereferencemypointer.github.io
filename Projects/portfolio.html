Three

This project contains some of the groundwork for a 3D game engine, using WebGL through three.js. This project is currently on permament pause. Functionality currently includes loading a model and playing an animation on demand, although this does not work in all browsers. There is a game object system that is not fully in use that handles models, animations, and behaviors, although the hypothetical behaviors currently supported are limited to motion and playing animations.
