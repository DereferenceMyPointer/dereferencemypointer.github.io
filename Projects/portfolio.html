Korval's Curse

Created in two weeks for the UW Game Dev Club Winter Jam 2021, this project is a 2D top-down game with roguelike mechanics. It has dynamic soundtracking, a flexible wave system, items that can be added to any character and interact via hit, kill, heal, death, etc. The enemies also have state-based behaviors that are editor friendly. I made the art, animations, and music in addition to architecting the game and writing all of the code during the two week time period I had to make it.

Three

This project contains some of the groundwork for a 3D game engine, using WebGL through three.js. This project is currently on permament pause. Functionality currently includes loading a model and playing an animation on demand, although this does not work in all browsers. There is a game object system that is not fully in use that handles models, animations, and behaviors, although the hypothetical behaviors currently supported are limited to motion and playing animations.
