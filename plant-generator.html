This is an L-system based procedural plant generator! The \"axiom\" is the initial command, and the \"rule\" is what you replace each letter of the branch with each iteration. Syntax is:

F: go forward by \"length\"

C or c: turn by \"angle\", counterclockwise and clockwise respectively

For rules: write the character to replace, followed by a comma, followed by what to replace it with. Open brackets are a new branch start and closed brackets are a branch end.
